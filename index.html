Setup Required

To use this site, you need a free YouTube Data API key.

  1. Go to Google Cloud Console
  2. Create a new project (or select existing)
  3. Search for & enable "YouTube Data API v3"
  4. Go to Credentials → Create Credentials → API Key
  5. Copy the key and paste it in config.js

The API key is free with daily quota of 10,000 requests.
